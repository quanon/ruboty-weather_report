# Ruboty::WeatherReport

A Ruboty handler to show weather information

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-weather_report'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-weather_report
